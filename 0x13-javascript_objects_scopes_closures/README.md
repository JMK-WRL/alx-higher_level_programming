# 0x13. JavaScript Objects, Scopes, Closures

This repository contains a series of JavaScript exercises focused on object-oriented programming, scopes, and closures. Below is an overview of each task along with the corresponding files.

## Task 0: Rectangle #0

**File:** [0-rectangle.js](./0x13-javascript_objects_scopes_closures/0-rectangle.js)

**Description:** Defines an empty class `Rectangle`.

## Task 1: Rectangle #1

**File:** [1-rectangle.js](./0x13-javascript_objects_scopes_closures/1-rectangle.js)

**Description:** Defines a class `Rectangle` with a constructor that initializes width and height attributes.

## Task 2: Rectangle #2

**File:** [2-rectangle.js](./0x13-javascript_objects_scopes_closures/2-rectangle.js)

**Description:** Defines a class `Rectangle` with a constructor that initializes width and height attributes. If either `width` or `height` is 0 or not a positive integer, an empty object is created.

## Task 3: Rectangle #3

**File:** [3-rectangle.js](./0x13-javascript_objects_scopes_closures/3-rectangle.js)

**Description:** Defines a class `Rectangle` with a constructor that initializes width and height attributes. An instance method, `print()`, is created to print the rectangle using the character 'X'.

## Task 4: Rectangle #4

**File:** [4-rectangle.js](./0x13-javascript_objects_scopes_closures/4-rectangle.js)

**Description:** Defines a class `Rectangle` with a constructor that initializes width and height attributes. Instance methods `print()`, `rotate()`, and `double()` are created.

## Task 5: Square #0

**File:** [5-square.js](./0x13-javascript_objects_scopes_closures/5-square.js)

**Description:** Defines a class `Square` that inherits from `Rectangle`. The constructor takes one argument, `size`, and calls the parent constructor using `super()`.

## Task 6: Square #1

**File:** [6-square.js](./0x13-javascript_objects_scopes_closures/6-square.js)

**Description:** Defines a class `Square` that inherits from `Square` with an instance method `charPrint(c)` to print the square using the character `c` or 'X' if `c` is undefined.

## Task 7: Occurrences

**File:** [7-occurrences.js](./0x13-javascript_objects_scopes_closures/7-occurrences.js)

**Description:** Contains a function `nbOccurences` that returns the number of occurrences of a search element in a list.

## Task 8: Esrever

**File:** [8-esrever.js](./0x13-javascript_objects_scopes_closures/8-esrever.js)

**Description:** Contains a function `esrever` that returns the reversed version
